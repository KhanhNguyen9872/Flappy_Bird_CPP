# Build for Linux
```bash
rm -rf Flappy_Bird_CPP 2>/dev/null
git clone https://github.com/KhanhNguyen9872/Flappy_Bird_CPP
cd Flappy_Bird_CPP
```

```bash
make linux # for linux
```

```bash
make termux # for termux
```

```bash
./build/flappy # run
```
